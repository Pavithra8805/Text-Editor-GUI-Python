# Quick Start Guide

Get started with Modern Text Editor in under 2 minutes.

---

## Installation

### Step 1: Check Python
Open terminal/command prompt and run:
```bash
python --version
```
You need Python 3.6 or higher.

### Step 2: Run the Editor
Navigate to the project folder:
```bash
cd "d:\Projects\ProjectHUB_Sources\270 Python Projects\Test Editor GUI"
```

Run the application:
```bash
python main.py
```

---

## First Steps

### Create Your First File

1. The editor opens with a blank document
2. Start typing immediately
3. Press Ctrl+S to save
4. Choose a location and filename
5. Click Save

### Open an Existing File

1. Press Ctrl+O
2. Navigate to your file
3. Select and click Open
4. Start editing

### Basic Editing

- Type normally as in any text editor
- Use mouse to select text
- Right-click for context menu (copy/paste)
- Use keyboard shortcuts for faster workflow

---

## Essential Shortcuts

```
Ctrl+N    New file
Ctrl+O    Open file
Ctrl+S    Save file
Ctrl+Z    Undo
Ctrl+Y    Redo
Ctrl+C    Copy
Ctrl+V    Paste
Ctrl+A    Select All
```

## Tips for Best Experience

1. Use keyboard shortcuts for faster workflow
2. Check status bar for current position
3. Save frequently (Ctrl+S)
4. Use zoom controls for comfortable viewing
5. Enable file auto-save prompts (already enabled)

---

## Common Tasks

### Creating Python Files
1. Create new file (Ctrl+N)
2. Write your Python code
3. Save as .py file (Ctrl+Shift+S)
4. The editor will detect Python file type

### Creating Web Files
1. Create new file
2. Save with .html, .css, or .js extension
3. File type shows in status bar
4. Continue editing

### Editing Multiple Files
1. Save current file
2. Open new file (Ctrl+O)
3. Edit as needed
4. Switch between files by opening them
